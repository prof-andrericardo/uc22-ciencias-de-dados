## 📚 Sobre o 2º Trimestre

O segundo trimestre da Unidade Curricular 22 – Ciências de Dados tem como foco principal **a coleta, análise, visualização e limpeza de dados reais**, utilizando a linguagem Python e a biblioteca Pandas.

Os conteúdos são aplicados de forma didática e progressiva, com atividades práticas baseadas em situações do cotidiano e projetos com dados reais (como os Pokémons).

## 🎯 Competências Desenvolvidas

- Compreensão da estrutura de arquivos CSV
- Leitura e manipulação de dados com Pandas
- Criação de filtros e análises descritivas
- Limpeza e padronização de dados inconsistentes
- Criação de gráficos informativos com Matplotlib
- Aplicação de conceitos de Ciência de Dados em contextos lúdicos e reais
